# Changelog

## v0.2.0 - 2026-06-24

### Added
- 参考图生成现在支持最多 4 张图片上传，并允许多选后继续追加。

### Changed
- 编辑原图模式继续保持单张上传，并与参考图生成模式分离为独立的上传链路。
- 页面左上角品牌区更新为新版图标加中英文名称的组合展示。
- favicon 更新为新的小图标资源。

## v0.1.2 - 2026-06-24

### Changed
- 将项目品牌统一更新为“魔法画布 (The Magic Canvas)”。
- 页面头部改为展示品牌 logo，并新增底部版权信息与项目仓库链接。
- README、页面标题和 favicon 同步切换为新的品牌图标与名称。

## v0.1.1 - 2026-06-24

### Changed
- 将整体界面调整为夏日清新风格，统一替换为更清爽的蓝绿与浅白配色。
- 优化背景、面板、按钮、上传区、历史卡片和预览弹层的视觉层次，让页面观感更轻盈凉爽。

## v0.1.0 - 2026-06-24

### Added
- 为结果预览区域新增点击图片打开弹层预览的能力。
- 在预览弹层中提供放大、缩小、1:1 展示和关闭操作。

### Changed
- 预览弹层默认按可视区域自适应展示图片，并在放大后通过滚动查看超出区域。
- 当生成记录尺寸不可用时，预览会自动读取图片真实像素尺寸用于显示和缩放计算。
