# Changelog

## v1.2.3 - 2026-07-01

### Fixed
- 修复 Gemini 图片生成通过 sub2api 中转时报 “Images API is not supported for this platform” 的问题，非 Google 官方域名改走 `/v1beta/models/{model}:generateContent` 并使用 Gemini `contents`/`generationConfig.imageConfig` 请求格式。

## v1.2.2 - 2026-07-01

### Changed
- 生图模型选择器仅展示 `gemini-3.1-flash-image` 和 `gpt-image-2`，辅助模型继续保留在设置页用于提示词相关能力。
- 侧边栏版本号改为读取当前构建版本，避免发布后页面显示旧版本号。

### Fixed
- 修正旧设置中选中过辅助模型时，主生图表单可能保留不可生图模型的问题。

## v1.2.1 - 2026-07-01

### Fixed
- 修复 Gemini 图片生成在 OpenAI 兼容中转接口上返回 404 的问题，非 Google 官方域名会在原生 Gemini 路径不可用时自动回退到 `/v1/images/generations`。

## v1.2.0 - 2026-06-30

### Added
- 接口设置页支持配置多个接口，可分别填写接口域名、API Key，并为每个接口多选支持的模型。
- 生成流程会根据当前选择的模型自动匹配对应接口配置，支持 `gemini-3.1-flash-image`、`gpt-image-2`、`gpt-5.5`、`gpt-5.4` 和 `gpt-5.4-mini`。
- Gemini 图片生成请求适配 REST 格式，使用 `response_format.aspect_ratio` 和 `response_format.image_size` 传递比例与生成尺寸。

### Changed
- 将图片比例与生成尺寸拆分为独立选项，2K 和 4K 不再混在比例预设中。
- 移除页面中的生成数量设置，生成接口固定按单张图片请求。

### Fixed
- 修复图生图上传区“移除”按钮实际触发加图的问题，改为每张图片右上角的小叉号删除，并保留共用的添加图片入口。

## v1.1.0 - 2026-06-28

### Added
- 为生成结果区和历史记录新增“继续编辑”快捷按钮，可直接将已生成图片带回图片编辑模式，无需手动下载再上传。
- 为尺寸预设新增 2K 和 4K 快捷按钮，便于快速选择更高分辨率输出。

### Changed
- 将前端生成质量选项补齐为自动、快速、标准、高清 4 档，与 `gpt-image-2` 当前支持的质量参数保持一致。

## v1.0.1 - 2026-06-28

### Added
- 新增两套风格预设缩略图生成产物，包含 6 个预设对应的 PNG 与生成元数据，便于筛选卡片视觉素材。

### Changed
- 调整写实、插画、动漫、3D 渲染和赛博朋克预设的提示词模板，使风格引导更通用，并更强调构图、光影、层次与材质表现。

## v0.2.0 - 2026-06-24

### Added
- 参考图生成现在支持最多 4 张图片上传，并允许多选后继续追加。

### Changed
- 编辑原图模式继续保持单张上传，并与参考图生成模式分离为独立的上传链路。
- 页面左上角品牌区更新为新版图标加中英文名称的组合展示。
- favicon 更新为新的小图标资源。

## v0.1.2 - 2026-06-24

### Changed
- 将项目品牌统一更新为“魔法画布 (The Magic Canvas)”。
- 页面头部改为展示品牌 logo，并新增底部版权信息与项目仓库链接。
- README、页面标题和 favicon 同步切换为新的品牌图标与名称。

## v0.1.1 - 2026-06-24

### Changed
- 将整体界面调整为夏日清新风格，统一替换为更清爽的蓝绿与浅白配色。
- 优化背景、面板、按钮、上传区、历史卡片和预览弹层的视觉层次，让页面观感更轻盈凉爽。

## v0.1.0 - 2026-06-24

### Added
- 为结果预览区域新增点击图片打开弹层预览的能力。
- 在预览弹层中提供放大、缩小、1:1 展示和关闭操作。

### Changed
- 预览弹层默认按可视区域自适应展示图片，并在放大后通过滚动查看超出区域。
- 当生成记录尺寸不可用时，预览会自动读取图片真实像素尺寸用于显示和缩放计算。
